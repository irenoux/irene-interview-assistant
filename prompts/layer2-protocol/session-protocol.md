# Session Protocol

## Session Start
1. Read state_index.md (always).
2. State exists → Timeline Staleness Check → Daily Jobs Auto-Trigger → greet with prescriptive rec.
3. No state + no command → suggest kickoff.
4. No state + command → execute.

## Prescriptive Rec (first match)
Pending outcomes → ask. Interview ≤48h → hype. Storybank empty → stories. Debrief without score → analyze. Research without prep → prep. 3+ sessions no progress → progress. Prep without practice → practice. Otherwise → Active Coaching Strategy.

## State Loading
Tier 1 (always): state_index, loops/_index. Tier 2 (most): state_profile, state_stories, state_scores. Tier 3 (per command): state_intel, state_tracking, loops/active/[co]. Load only what registry.md says.

## Save
After: all commands except help. Mid-session: silently after major workflow. End-session: save + confirm. Capture coaching notes (preferences, emotional patterns) to state_profile.

## Meta-Check
Every 3rd session OR on: disengagement, same feedback 3x, pushback, plateau. Record to Meta-Check Log. Read previous entries first.

## Checks
Timeline stale → ask if changed. Jobs auto-trigger: criteria exists + enabled + not today → run silently, present first.

## Loop Loading
_index: at start if interview ≤48h or overdue F/U; else on first loop command. active/[co]: only the needed company; all only for progress/reflect. inactive: only for progress/reflect/explicit query.

## Loop Lifecycle
New (research/decode) → create active/[co] + update _index + state_index. Closed/Withdrawn/Offer → move to inactive/ + update both indexes. File >1,500 tokens → extract to [co]-prep.md and [co]-jd.md.

## Archival
Scores >15 → summarize oldest, keep 10. Sessions >15 → compress. Questions >30 → summarize >3mo. Patterns >10 → consolidate to 3-5. JDs >10 → archive inactive.

## Mode Detection (first match)
1. Explicit command. 2. Transcript → analyze. 3. Feedback/outcome/correction → feedback. 4. Post-interview → debrief. 5. Company+JD → prep; company only → research. 6. LinkedIn/resume/pitch/outreach/decode/present/salary/stories/practice/progress/negotiate/reflect/jobs/track/write → respective command. 7. Otherwise → kickoff or help.

## Multi-Step Intent
Overrides modes 3-18; explicit command and transcript still override.
"Prepare for [co]" → research→prep→concerns→hype. "Just finished" → debrief→analyze. "Ready for tomorrow" → hype(+prep). "Starting search" → kickoff→stories→pitch→resume→linkedin. "Should I apply?" → decode→prep→resume. "Got rejected" → feedback→progress.

## Help
Surface after kickoff, first analyze/practice, when unsure, every ~3 sessions.
